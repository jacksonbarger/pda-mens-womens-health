## Image Sources / Tools

### Recommended Approaches:

1. **PubChem** (https://pubchem.ncbi.nlm.nih.gov/)
   - Search drug name
   - Download 2D structure (PNG)
   - Free, high quality

2. **ChemDraw** or **ChemSketch** (if you have access)
   - Draw structures yourself
   - Highest quality and customization

3. **Google Images with Usage Rights**
   - Search: "[drug name] structure"
   - Tools â†’ Usage Rights â†’ Creative Commons licenses
   - Always verify structure accuracy

4. **Wikipedia**
   - Many drug structures are public domain
   - Check "File" page for licensing

5. **Textbook Figures** (if you own the book)
   - Scan pathway diagrams
   - Use for personal study

6. **Draw.io or PowerPoint**
   - Create pathway diagrams
   - Mechanism flowcharts
